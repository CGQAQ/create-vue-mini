## 使用

要使用 `create-vue-mini` 创建一个全新的 Vue Mini 小程序项目，请直接在终端内运行以下命令：

```bash
npm create vue-mini@latest
```

<!-- prettier-ignore -->
> [!NOTE]
> `@latest` **不能**省略, 否则 `npm` 可能会使用已缓存的旧版本.
